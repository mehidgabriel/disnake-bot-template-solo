# Discord Bot Template
Discord Bot template using Disnake library

## Installation
```bash
git clone https://github.com/mehidgabriel/disnake-bot-template
cd disnake-bot-template
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Usage
```bash
python3 bot.py
```
